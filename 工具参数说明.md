## 工具说明

### 1. 图像理解 (understand\_image)

分析图像内容，支持多种输入方式。

**参数：**

- `images` (string\[]): 图像数组，每个元素可以是 URL、Base64 编码、本地文件路径或 localhost URL
- `question` (string, 可选): 关于图像的问题或指令（默认："Please describe the content of this image"）
- `max_tokens` (number, 可选): 最大响应 token 数（1-4096，默认：1024）

**示例：**

```json
{
  "images": ["https://example.com/image.png"],
  "question": "描述这张图片的内容",
  "max_tokens": 1024
}
```

### 2. 音频理解 (understand\_audio)

分析音频内容，支持多种输入方式。

**参数：**

- `audio` (string): 音频 URL、Base64 编码、本地文件路径或 localhost URL
- `question` (string, 可选): 关于音频的问题或指令（默认："Please describe the content of this audio"）
- `max_tokens` (number, 可选): 最大响应 token 数（1-4096，默认：1024）

**示例：**

```json
{
  "audio": "https://example.com/audio.wav",
  "question": "描述这段音频的内容",
  "max_tokens": 1024
}
```

### 3. 视频理解 (understand\_video)

分析视频内容，支持多种输入方式。

**参数：**

- `video` (string): 视频 URL、Base64 编码、本地文件路径或 localhost URL
- `question` (string, 可选): 关于视频的问题或指令（默认："Please describe the content of this video"）
- `max_tokens` (number, 可选): 最大响应 token 数（1-4096，默认：1024）

**示例：**

```json
{
  "video": "https://example.com/video.mp4",
  "question": "描述视频中发生了什么",
  "max_tokens": 1024
}
```

### 4. 语音识别 (speech\_recognition)

将音频转换为文本。

**参数：**

- `audio` (string): Base64 编码的音频数据（带 MIME 前缀）或本地文件路径
- `language` (string, 可选): 识别语言（"auto"、"zh"、"en"，默认："auto"）

**示例：**

```json
{
  "audio": "data:audio/wav;base64,UklGR...",
  "language": "zh"
}
```

### 5. 语音合成 (speech\_synthesis)

使用预置音色将文本转换为语音。

**参数：**

- `text` (string): 要转换的文本（1-5000 字符）
- `voice` (string, 可选): 预置音色 ID（默认："mimo\_default"）
- `style_instruction` (string, 可选): 风格指令
- `output_format` (string, 可选): 输出格式（"wav" 或 "mp3"，默认："wav"）
- `output_path` (string): 输出文件路径

**预置音色：**

- `mimo_default`: MiMo 默认音色
- `冰糖`: 中文女性
- `茉莉`: 中文女性
- `苏打`: 中文男性
- `白桦`: 中文男性
- `Mia`: 英文女性
- `Chloe`: 英文女性
- `Milo`: 英文男性
- `Dean`: 英文男性

**示例：**

```json
{
  "text": "今天天气真好！",
  "voice": "茉莉",
  "style_instruction": "用明亮活泼的语调",
  "output_format": "wav",
  "output_path": "./output.wav"
}
```

### 6. 音色设计 (voice\_design)

通过文本描述自定义音色特征。

**参数：**

- `text` (string): 要转换的文本（1-5000 字符）
- `voice_description` (string): 音色特征描述（10-500 字符）
- `output_format` (string, 可选): 输出格式（"wav" 或 "mp3"，默认："wav"）
- `optimize_text_preview` (boolean, 可选): 是否智能润色文本（默认：false）
- `output_path` (string): 输出文件路径

**音色描述示例：**

- "年轻女性，温暖自信"
- "深沉沙哑的声音"
- "五十多岁的中年男性"
- "温柔治愈系女声"

**示例：**

```json
{
  "text": "今天天气真好！",
  "voice_description": "年轻女性，温暖自信",
  "output_format": "wav",
  "optimize_text_preview": true,
  "output_path": "./output.wav"
}
```

### 7. 音色克隆 (voice\_clone)

基于音频样本克隆音色。

**参数：**

- `text` (string): 要转换的文本（1-5000 字符）
- `voice_audio` (string): 音频样本（Base64 编码或本地文件路径）
- `style_instruction` (string, 可选): 风格指令
- `output_format` (string, 可选): 输出格式（"wav" 或 "mp3"，默认："wav"）
- `output_path` (string): 输出文件路径

**示例：**

```json
{
  "text": "今天天气真好！",
  "voice_audio": "./sample.wav",
  "style_instruction": "用温柔的语调",
  "output_format": "wav",
  "output_path": "./output.wav"
}
```
